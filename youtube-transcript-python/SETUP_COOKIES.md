# Hướng dẫn Setup Cookies để Bypass Bot Detection

## Quick Start (Cách nhanh nhất)

1. **Lấy cookies từ Chrome:**
   - Mở Chrome → YouTube → Login
   - Nhấn `F12` → Application → Cookies → `https://www.youtube.com`
   - Copy các cookies: `VISITOR_INFO1_LIVE`, `YSC`, `PREF` (copy Name và Value)

2. **Dùng script để convert:**
   ```bash
   cd youtube-transcript-python
   python3 convert_cookies.py
   # Nhập: VISITOR_INFO1_LIVE|paste-value-here
   # Nhập: YSC|paste-value-here
   # Nhập: PREF|paste-value-here
   # Enter để kết thúc
   ```

3. **Encode và deploy:**
   ```bash
   cat cookies.txt | base64
   # Copy output và set trên Cloud Run
   gcloud run services update youtube-transcript-python \
     --set-env-vars COOKIES_BASE64="<paste-base64-here>" \
     --region asia-southeast1
   ```

## Tại sao cần cookies?

YouTube thường chặn requests từ Cloud Run vì:
- IP addresses của Cloud Run bị đánh dấu là "cloud IPs"
- Không có cookies = không có session = dễ bị detect là bot

**Cookies là giải pháp hiệu quả nhất** để bypass bot detection!

## Bước 1: Export Cookies từ Browser (Thủ công)

### Cách 1: Dùng Chrome DevTools (Khuyên dùng)

1. **Login vào YouTube:**
   - Mở Chrome và vào [YouTube](https://www.youtube.com)
   - Đảm bảo bạn đã login vào tài khoản Google/YouTube

2. **Mở DevTools:**
   - Nhấn `F12` hoặc `Cmd+Option+I` (Mac) / `Ctrl+Shift+I` (Windows/Linux)
   - Vào tab **Application** (hoặc **Storage** trong Firefox)

3. **Lấy cookies:**
   - Trong sidebar bên trái, mở **Cookies** → `https://www.youtube.com`
   - Bạn sẽ thấy danh sách tất cả cookies

4. **Tạo file cookies.txt:**
   
   Tạo file `cookies.txt` với format Netscape cookies. File phải có format như sau:
   
   ```
   # Netscape HTTP Cookie File
   # This file was generated by hand
   .youtube.com	TRUE	/	FALSE	1735689600	VISITOR_INFO1_LIVE	abc123...
   .youtube.com	TRUE	/	FALSE	1735689600	YSC	xyz789...
   .youtube.com	TRUE	/	FALSE	1735689600	PREF	f1=...
   www.youtube.com	FALSE	/	TRUE	1735689600	LOGIN_INFO	AFmmF2...
   ```
   
   **Format:** `domain	includeSubdomains	path	secure	expiration	name	value`
   
   - `domain`: Domain của cookie (ví dụ: `.youtube.com` hoặc `www.youtube.com`)
   - `includeSubdomains`: `TRUE` nếu cookie áp dụng cho subdomains, `FALSE` nếu không
   - `path`: Path của cookie (thường là `/`)
   - `secure`: `TRUE` nếu chỉ gửi qua HTTPS, `FALSE` nếu không
   - `expiration`: Timestamp Unix (seconds) khi cookie hết hạn
   - `name`: Tên cookie
   - `value`: Giá trị cookie

5. **Copy cookies quan trọng:**
   
   Trong DevTools, bạn sẽ thấy bảng với các cột: Name, Value, Domain, Path, Expires, etc.
   
   **Các cookies quan trọng nhất cần copy:**
   - `VISITOR_INFO1_LIVE` - Cookie này rất quan trọng!
   - `YSC` - YouTube session cookie
   - `PREF` - Preferences cookie
   - `LOGIN_INFO` - Login info (nếu có, thường ở domain `www.youtube.com`)
   - `__Secure-3PSID` - Secure session ID (nếu có)
   - `__Secure-3PAPISID` - Secure API session ID (nếu có)
   - Bất kỳ cookie nào có tên bắt đầu bằng `__Secure-`
   
   **Cách copy:**
   - Click vào từng cookie trong danh sách
   - Copy giá trị từ cột **Value** (đây là phần quan trọng nhất)
   - Ghi lại **Name** của cookie
   - Ghi lại **Domain** (thường là `.youtube.com` hoặc `www.youtube.com`)
   - Ghi lại **Secure** (có checkbox hay không)
   
   **Ví dụ:**
   ```
   Name: VISITOR_INFO1_LIVE
   Value: abc123def456ghi789... (copy cái này)
   Domain: .youtube.com
   Path: /
   Secure: ✓ (checked)
   ```

6. **Tạo file cookies.txt:**
   
   **Cách A: Dùng script (Khuyên dùng - xem Cách 2 ở trên)**
   
   **Cách B: Tạo thủ công:**
   
   Tạo file `cookies.txt` và thêm từng cookie theo format trên. Ví dụ:
   
   ```bash
   # Tạo file cookies.txt
   cat > cookies.txt << 'EOF'
   # Netscape HTTP Cookie File
   # This file was generated by hand
   .youtube.com	TRUE	/	FALSE	9999999999	VISITOR_INFO1_LIVE	<paste-value-here>
   .youtube.com	TRUE	/	FALSE	9999999999	YSC	<paste-value-here>
   .youtube.com	TRUE	/	FALSE	9999999999	PREF	<paste-value-here>
   www.youtube.com	FALSE	/	TRUE	9999999999	LOGIN_INFO	<paste-value-here>
   EOF
   ```
   
   **Lưu ý:** 
   - Thay `<paste-value-here>` bằng giá trị thực từ DevTools (cột "Value")
   - Dùng `9999999999` cho expiration (tương đương năm 2286, đủ xa)
   - Dùng **tab** (`\t`) để phân cách các cột, KHÔNG phải spaces
   - Format: `domain[TAB]includeSubdomains[TAB]path[TAB]secure[TAB]expiration[TAB]name[TAB]value`
   
   **Cách kiểm tra format đúng:**
   ```bash
   # Xem file với tabs hiển thị
   cat -A cookies.txt
   # Nếu thấy ^I thì đó là tab (đúng)
   # Nếu thấy nhiều spaces thì sai
   ```

### Cách 2: Dùng Script Python (Dễ nhất - Khuyên dùng)

Script `convert_cookies.py` sẽ giúp bạn convert cookies dễ dàng:

1. **Chạy script:**
   ```bash
   cd youtube-transcript-python
   python3 convert_cookies.py
   ```

2. **Nhập cookies:**
   - Script sẽ hỏi bạn nhập cookies
   - Format: `name|value` (đơn giản nhất)
   - Hoặc: `domain|name|value|path|secure|expiration_days` (đầy đủ)
   
   Ví dụ:
   ```
   VISITOR_INFO1_LIVE|abc123def456...
   YSC|xyz789...
   PREF|f1=50000000...
   ```

3. **Script sẽ tự động:**
   - Tạo file `cookies.txt` với format đúng
   - Set domain, path, expiration mặc định
   - Hiển thị preview

**Lưu ý:** Lấy giá trị cookies từ Chrome DevTools:
- Mở DevTools (F12) → Application → Cookies → https://www.youtube.com
- Click vào từng cookie để xem Name và Value
- Copy Value và paste vào script

### Cách 3: Dùng Firefox DevTools

1. Mở Firefox và login vào YouTube
2. Nhấn `F12` để mở DevTools
3. Vào tab **Storage** → **Cookies** → `https://www.youtube.com`
4. Copy cookies và format tương tự như trên

## Bước 3: Set Environment Variable trên Cloud Run

### Cách 1: Dùng gcloud CLI (Nhanh nhất)

```bash
# Thay <YOUR_BASE64_COOKIES> bằng output từ bước 2
gcloud run services update youtube-transcript-python \
  --set-env-vars COOKIES_BASE64="<YOUR_BASE64_COOKIES>" \
  --region asia-southeast1 \
  --project YOUR_PROJECT_ID
```

### Cách 2: Dùng Google Cloud Console

1. Vào [Cloud Run Console](https://console.cloud.google.com/run)
2. Click vào service `youtube-transcript-python`
3. Click "Edit & Deploy New Revision"
4. Vào tab "Variables & Secrets"
5. Click "Add Variable"
6. Name: `COOKIES_BASE64`
7. Value: Paste base64 string từ bước 2
8. Click "Deploy"

### Cách 3: Dùng Secret Manager (Bảo mật nhất)

```bash
# Tạo secret
echo -n "<YOUR_BASE64_COOKIES>" | gcloud secrets create youtube-cookies-base64 --data-file=-

# Grant access (thay <SERVICE_ACCOUNT> bằng service account của Cloud Run)
gcloud secrets add-iam-policy-binding youtube-cookies-base64 \
  --member="serviceAccount:<SERVICE_ACCOUNT>@<PROJECT_ID>.iam.gserviceaccount.com" \
  --role="roles/secretmanager.secretAccessor"

# Mount secret
gcloud run services update youtube-transcript-python \
  --set-secrets COOKIES_BASE64=youtube-cookies-base64:latest \
  --region asia-southeast1
```

## Lưu ý

1. **Cookies có thể expire:** Nếu cookies hết hạn, bạn cần export lại và update
2. **Không share cookies:** Cookies chứa session của bạn, không nên share công khai
3. **Refresh định kỳ:** Nên refresh cookies mỗi vài tuần để đảm bảo hoạt động tốt

## Troubleshooting

### Vẫn bị lỗi bot detection?

1. **Kiểm tra cookies có hợp lệ không:**
   ```bash
   # Decode và xem cookies
   echo "<YOUR_BASE64_COOKIES>" | base64 -d | head
   ```

2. **Kiểm tra logs:**
   ```bash
   gcloud logging read "resource.type=cloud_run_revision AND resource.labels.service_name=youtube-transcript-python" --limit 50
   ```

3. **Thử tăng số retries:**
   ```bash
   gcloud run services update youtube-transcript-python \
     --set-env-vars MAX_RETRIES=5 \
     --region asia-southeast1
   ```

4. **Export cookies lại:** Có thể cookies đã hết hạn, export lại từ browser

## Alternative: Dùng Proxy

Nếu không muốn dùng cookies, có thể thử proxy:

```bash
gcloud run services update youtube-transcript-python \
  --set-env-vars PROXY_URL="http://proxy-host:port" \
  --region asia-southeast1
```

Tuy nhiên, cookies vẫn là giải pháp hiệu quả nhất!
